Welcome to LoopSocket

LoopSocket is a collaborative loop based sequencer/drum machine

  • Click Sign Up on the top-right corner to get started
  • Go to Loops to create a new session or view/edit existing sesssions
  • Add other members to your loop or make them public to collaborate.
  • Public and shared loops can be viewed and edited by others but only the owner can save them, so just be sure to save before making your loops public
  • Have fun!
